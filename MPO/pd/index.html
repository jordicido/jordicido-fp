    Programación didáctica MPO Febrero 2025 Prometeo¶

    Unidades de Trabajo (UT)¶

    UT1: Fundamentos de Python (Trabajado anteriormente)¶

    Duración: 8 semanas Objetivos: - Comprender los conceptos básicos de programación - Manejar variables y tipos de datos básicos - Utilizar operadores y estructuras básicas de entrada/salida

    Contenidos: - Variables: declaración, inicialización y asignación - Tipos de datos primitivos: int, float, bool, str - Operadores aritméticos, lógicos y de comparación - Entrada y salida: input() y print() - Comentarios y documentación del código - Buenas prácticas de programación

    UT2: Estructuras de Control¶

    Duración: 4 semanas (Semanas 9, 10, 12) Objetivos: - Implementar estructuras condicionales para la toma de decisiones - Utilizar bucles para repetir procesos - Combinar estructuras de control para resolver problemas complejos

    Contenidos: - Estructuras condicionales: if, elif, else - Operadores lógicos y expresiones booleanas - Bucle for: iteración sobre secuencias - Bucle while: repetición condicional - Control de flujo: break, continue, pass - Estructuras anidadas y buenas prácticas

    UT3: Estructuras de Datos¶

    Duración: 2 semanas (Semanas 14, 15) Objetivos: - Manejar colecciones de datos de forma eficiente - Elegir la estructura de datos adecuada según el problema - Manipular y procesar información estructurada

    Contenidos: - Listas: creación, indexado, métodos principales - Tuplas: inmutabilidad y casos de uso - Diccionarios: pares clave-valor, métodos de acceso - Operaciones comunes: ordenación, búsqueda, filtrado - Comprensión de listas básica - Conversión entre tipos de datos

    UT4: Funciones y Modularidad¶

    Duración: 4 semanas (Semanas 16, 17, 18, 19) Objetivos: - Crear código reutilizable mediante funciones - Organizar el código en módulos - Aplicar conceptos de alcance y parámetros

    Contenidos: - Definición y llamada de funciones - Parámetros y argumentos: posicionales, nombrados, por defecto - Valores de retorno y la sentencia return - Alcance de variables: local vs global - Funciones lambda básicas - Módulos: importación y creación - Documentación de funciones (docstrings) - Debugging y testing básico

    UT5: Manejo de Archivos y Errores¶

    Duración: 3 semanas (Semanas 20, 21, 22) Objetivos: - Leer y escribir archivos de diferentes formatos - Gestionar errores de forma controlada - Procesar archivos de logs y configuración

    Contenidos: - Apertura y cierre de archivos: open(), close() - Modos de apertura: lectura, escritura, anexado - Lectura de archivos: readline(), readlines(), read() - Escritura de archivos: write(), writelines() - Gestión de rutas y directorios - Manejo de excepciones: try, except, finally - Tipos de errores comunes en archivos - Procesamiento de archivos CSV básico

    UT6: Automatización de Tareas del Sistema¶

    Duración: 2 semanas (Semanas 23, 24) Objetivos: - Crear scripts para automatizar tareas administrativas - Interactuar con el sistema operativo - Manejar fechas y horarios en scripts

    Contenidos: - Módulo os: navegación de directorios, gestión de archivos - Módulo sys: argumentos de línea de comandos - Ejecución de comandos del sistema - Módulo datetime: fechas, horas, cálculos temporales - Programación de tareas básica - Scripts de mantenimiento: limpieza, backup - Variables de entorno y configuración

    UT7: Conectividad y Datos¶

    Duración: 2 semanas (Semanas 25, 26) Objetivos: - Consumir APIs web básicas - Procesar datos en formatos JSON y CSV - Implementar comunicación de red simple

    Contenidos: - Módulo requests: peticiones HTTP GET/POST - Trabajo con APIs REST básicas - Formato JSON: lectura y escritura - Procesamiento de archivos CSV con pandas básico - Descarga y procesamiento de datos - Autenticación básica en APIs - Logs de red y monitorización

    UT8: Proyecto Integrador¶

    Duración: 3 semanas (Semanas 27, 28, 29) Objetivos: - Aplicar todos los conocimientos adquiridos - Desarrollar un proyecto completo de automatización - Presentar y documentar el trabajo realizado

    Contenidos: - Planificación y diseño del proyecto - Análisis de requisitos para automatización - Desarrollo incremental del script - Testing y depuración avanzada - Documentación técnica del proyecto - Presentación de resultados - Casos de uso reales en administración de sistemas - Evaluación y mejora continua

    Proyectos sugeridos: - Script de backup automatizado - Monitor de logs del sistema - Generador de informes automáticos - Herramienta de gestión de usuarios - Sistema de notificaciones automáticas

    Temporalización¶

    Trabajado anteriormente¶

    • Variables: declaración, inicialización.
    • Tipos de datos: enteros, flotantes, booleanos, cadenas de texto.
    • Entrada y salida de datos: input(), print().
    • Operadores: aritméticos, lógicos, de comparación.
    • Comentarios en el código: una línea, varias líneas.
